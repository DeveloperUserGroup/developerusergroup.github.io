Sponsorship

We limit to 4 key sponsors of the Developer User Group. Our key sponsors gain the following privileges:

  • Regular tweets of sponsor activities
  • Invites to speaker dinners.
  • Sponsor logo on the sponsor slide displayed to the group before the event starts
  • Sponsor pull up banner placed in the event room
  • Sponsor thanked by committee at each event
  • Sponsor logo displayed on DeveloperUG website
Key sponsorship costs R35 000. If you are interested in sponsoring DevUG, please email us.

Employment / Vacancies

If you have employment positions available that you would like to fill we suggest you use Stack Overflow's careers site or OfferZen .

Alternatively you can sponsor swag for an event, the minimum value of the swag should be R5000 and should be no more than 5 items, at which point we will make brief mention of the position at our next meeting and advise a URL that interested members can visit. If you are interested in this option please contact the organizers.

Conferences / Courses

If you have a conference / course that you would like to advertise and the organizers deem it appropriate for the group we would require one ticket be made available to the group for a lucky draw at the next Meetup.

Alternatively you can sponsor swag for an event (the minimum value of the swag should be R5000 and should be no more than 5 items) at which point we will make brief mention of the event at our next meeting and advise a URL that interested members can visit. If you are interested in this option please contact the organizers.
