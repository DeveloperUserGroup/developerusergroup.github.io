Employment / Vacancies

If you have employment positions available that you would like to fill we suggest you use Stack Overflow's careers site or OfferZen.

Alternatively you can sponsor swag for an event, the minimum value of the swag should be R5000, at which point we will make brief mention of the position at our next meeting and advise a URL that interested members can visit. If you are interested in this option please contact the organizers.

Conferences / Courses

If you have a conference / course that you would like to advertise and the organizers deem it appropriate for the group we would require one ticket be made available to the group for a lucky draw at the next Meetup.

Alternatively you can sponsor swag for an event (the minimum value of the swag should be R5000) at which point we will make brief mention of the event at our next meeting and advise a URL that interested members can visit. If you are interested in this option please contact the organizers.
